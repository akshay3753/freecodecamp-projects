# Dr. Norman Borlaug Tribute Page

This is a responsive tribute page dedicated to **Dr. Norman Borlaug**, the agricultural scientist whose groundbreaking work in crop production saved millions of lives during the Green Revolution.

This project fulfills all the user stories required by freeCodeCamp's **"Build a Tribute Page"** challenge.

## 📜 Features

- Semantic HTML5 structure with `<main>`, `<figure>`, `<figcaption>`, and proper use of `alt` attributes for accessibility
- Responsive image that scales with screen size
- Accessible and well-labeled layout that meets WCAG basics
- External link to additional information with `target="_blank"`
- Clean CSS design mimicking the visual style of the freeCodeCamp example

## 🚀 Built With

- HTML5
- CSS3 (with `max-width`, `height: auto`, flexbox-based centering, and simple shadow effects)

## 📁 Project Structure

project-folder/ │ ├── index.html # Main HTML file ├── styles.css # Custom CSS styling └── README.md # This file



## 🧪 How to Run

1. Clone or download this repository.
2. Open `index.html` in your preferred browser.
3. Inspect the layout and interact with the link to learn more about Dr. Borlaug.
